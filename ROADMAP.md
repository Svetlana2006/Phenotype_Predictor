# Project Roadmap

This document outlines the planned future development for the Forensic Phenotype Predictor.

## Version 1.1
- **PostgreSQL Case Management**: Database infrastructure to organize predictions by case numbers.
- **PDF Report Generation**: Export professional, court-ready forensic reports detailing the phenotype prediction, evidence quality, and explainability metrics.

## Version 1.2
- **VCF Parser Support**: Expand the biological parser to ingest real sequencing files (`.vcf`), rather than just raw sequences.
- **JWT Authentication**: Role-based access control (Investigator, Researcher, Admin).

## Version 2.0
- **Docker Containerization**: Containerize the backend, frontend, and database for seamless on-premise enterprise deployment (Public cloud demo already completed ahead of schedule).
- **Immutable Audit Trails**: Permanent tracking of timestamps, model versions, confidence scores, and predictions for legal compliance.
- **Collaboration Analytics**: Dashboards for cross-agency case comparisons and data analytics.
